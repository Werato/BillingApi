## Legacy stack note

This project was originally created with ASP.NET Core 2.1 and is kept as a legacy-style demo project.

The goal of the project is to demonstrate basic backend architecture concepts:
- ASP.NET Core Web API
- MVC frontend
- service-to-service communication with HttpClient
- dependency injection
- payment gateway abstraction
- basic unit testing
- simplified billing flow

The project is not intended to represent a modern production-ready payment system.
